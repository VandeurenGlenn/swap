# Fox-Swap

Multichain Aggregator Swap

## Develop

### Before first run or after PULL

run

```sh
npm i
```

### Serve

make sure to install `@vandeurenglenn/project`

```sh
npm i -g @vandeurenglenn/project
```

run

```sh
npm run serve
```

### Watch

run

```sh
npm run watch
```

### Build

run

```sh
npm run build
```
